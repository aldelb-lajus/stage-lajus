🚀 Avant de commencer

Tu pars de zéro ? Parfait ! Suis ces étapes une par une, dans l'ordre. Si tu bloques sur une étape plus de 10 minutes, appelle ton tuteur. 🙂

  1. Installer VS Code (le logiciel pour écrire le code).
    VS Code, c'est comme un super bloc-notes fait exprès pour programmer.
    • Ouvre ton navigateur (Chrome, Edge ou Firefox).
    • Va sur le site : code.visualstudio.com
    • Clique sur le gros bouton bleu « Download for Windows ».
    • Un fichier se télécharge (en bas de l'écran). Double-clique dessus pour l'ouvrir.
    • Coche « J'accepte », puis clique Suivant à chaque étape, puis Installer, puis Terminer.

    💡 Cette installation ne demande pas de mot de passe administrateur. Si jamais on t'en demande un, appelle ton tuteur.

  2. Mettre le dossier du projet sur l'ordinateur.
    • Ton tuteur te donne le dossier du projet (clé USB ou lien).
    • Copie le dossier entier sur le Bureau : clic droit sur le dossier → Copier, puis sur le Bureau clic droit → Coller.
  3. Voir le site (le résultat).
    • Ouvre le dossier du projet (double-clic).
    • Double-clique sur le fichier index.html.
    • Ton navigateur s'ouvre et affiche le site 🌿 — c'est ce que tu vas améliorer !
  4. Ouvrir ton carnet de missions.
    • Dans le même dossier, double-clique sur missions.html.
    • Ton carnet s'ouvre (onglets Accueil, Déroulement, Aide). Laisse-le ouvert : c'est ton guide !
  5. Ouvrir le code dans VS Code.
    • Ouvre VS Code.
    • En haut : menu Fichier → Ouvrir le dossier...
    • Choisis le dossier du projet (sur le Bureau), puis Sélectionner un dossier.
    • À gauche apparaît la liste des fichiers : index.html, style.css, script.js... C'est là que tu vas écrire !
  6. Bien organiser ton écran.
    • Mets VS Code sur la moitié gauche et le navigateur (le site) sur la moitié droite.
    • Astuce : touche Windows + flèche gauche (ou droite) colle une fenêtre sur la moitié de l'écran.
  7. C'est parti !
    • Dans missions.html, va dans l'onglet Déroulement et commence la Partie 1.
    • Rappel magique : après chaque modification → Ctrl+S (sauvegarder), puis F5 dans le navigateur (rafraîchir) pour voir le résultat !
